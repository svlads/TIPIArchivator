### Архиватор (Задача 7)
Сборка:
``
mkdir build && cd build && cmake .. && make
``

Использование:\
`./arch encode in1 in2 in3` -- сжать файлы in1, in2, in3 и результат
поместить в файлы in1_comp, in2_comp, in3_comp и т д.

`./arch decode in1 in2 in3` -- восстановить файлы in1, in2, in3 и
результат записать в файлы in1_orig, in2_orig, in3_orig.

#### Описание алгоритма
Для сжатия файлов используется алгоритм Хаффмана. Перед сжатыми данными
в файл записываются символы с листьев дерева в порядке обхода в глубину.
Далее записывается само дерево в бинарном виде (1 - спуск вниз влево,
0 - поднимаемся вверх до первой вершины, у который мы в левом поддереве
и переходим в правое поддерево). Далее записывается размер оригинального
файла в байтах и закодированные данные.
